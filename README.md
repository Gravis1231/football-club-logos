# Football Club Logos

A comprehensive collection of **2382 football/soccer logos** in PNG format (700×700 px), organized by country.

## Structure

```
/{country}/{team-slug}.png
```

**Example:**
```
/england/manchester-united.png
/spain/barcelona.png
/germany/bayern-munchen.png
```

## Coverage

- 100+ countries and leagues
- Club teams, national teams, and tournaments
- 700×700 px transparent PNG format
- Source: [football-logos.cc](https://football-logos.cc)
